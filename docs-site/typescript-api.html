TypeScript API

The TypeScript SDK mirrors the Python engine at parity — the same journal, policy, dry-run, isolation, recovery, envelope, and compensators. The idiom is async/callback rather than a context manager.

Install

npm install pherix

Declare a tool and run a transaction

agentTxn(adapters, fn, options?) takes a callback. Effects journalled inside it commit when the callback resolves; throw or call ctx.rollback() to undo.

import { tool, agentTxn, SqliteAdapter, AuditJournal } from "pherix";

const insertUser = tool(
  { resource: "sql" },
  (db, name: string, role: string) => {
    db.exec("INSERT INTO users (name, role) VALUES (?, ?)", [name, role]);
    return name;
  },
);

const adapters = { sql: new SqliteAdapter(db) };

await agentTxn(adapters, async (txn) => {
  await insertUser("ada", "engineer");
  if (somethingWrong) await txn.rollback();   // undo the writes
});                                              // resolves cleanly → commit

Surface

SymbolWhat it is
tool(options, fn)Wrap a side-effecting function as a journalled tool.
agentTxn(adapters, fn, options?)Open a transaction around an async callback.
TxnContextThe txn argument — rollback(), commit(), approveIrreversible().
Policy, PolicyVerdictAllow/deny, caps, the human gate — evaluated at stage- and commit-time.
AuditJournalThe durable transcript; AuditJournal.inMemory() for tests.
dryRun, DryRunResultFold forward against a snapshot, then discard.
recover, RecoveryReportResume the backward fold after a crash from the durable journal.
isolation: Conflict, executeIsolatedRead/write-key conflict detection between concurrent transactions.
envelopeDurable cross-run spend caps.
compensatorsThe vetted semantic-inverse catalog (pherix/compensators).

Adapters

The TypeScript SDK ships the core reversible/irreversible adapters:

AdapterResourceReversible
SqliteAdapterSQLiteyes — savepoint
PostgresAdapterPostgreSQLyes — savepoint
FilesystemAdapterfilesyes — content backup
HttpAdapterHTTP callsno — staged / gated
The engine is at full parity with Python; the adapter catalog is broader on the Python side today (object stores, NoSQL, search). Any agent — in any language — can also reach the engine through the MCP gateway.
