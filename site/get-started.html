// get started

Run it free, or let us operate it.

The runtime is open source and free to self-host. The control plane — the operated, hard-to-self-run layer above it — is what you pay for, when you need it.

// quickstart · open source

1

Install

One package, no services to stand up.

$pip install pherix
2

Wrap your agent's tools

Keep your loop and model. Open a transaction; every tool call runs inside it.

from pherix import agent_txn

with agent_txn(adapters, policy) as txn:
    agent.run(task)        # tool calls journalled + snapshotted live
# commit on clean exit — any exception rolls the whole run back
3

That's it

Reversible effects roll back byte-clean; irreversible ones gate or compensate; the journal is your audit trail. See how it works for the model.
